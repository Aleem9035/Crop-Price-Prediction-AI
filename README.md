# 🌾 Crop Price Prediction System-AI

## Problem
Farmers don't know when to sell or store their crops — 
they lose money due to wrong timing.

## Solution
AI system that:
- Predicts future crop prices using ML models
- Gives smart recommendations (sell now or store?)
- Uses RAG + LangChain for intelligent responses

## Tech Stack
- ML Models: Prophet, Scikit-learn
- AI: RAG + LangChain + HuggingFace
- Backend: FastAPI
- Deployment: HuggingFace Spaces
